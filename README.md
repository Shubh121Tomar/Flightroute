# Flight Route Project

## Overview
This is a Java-based Flight Route Planner project that helps users find and manage flight routes efficiently.

## Features
- Add and manage flight information
- Find flight routes between destinations
- Uses custom data structures
- Efficient route processing

## Technologies Used
- Java
- Queue (ArrayQueue)
- Heap
- Object-Oriented Programming (OOP)

## Project Structure
